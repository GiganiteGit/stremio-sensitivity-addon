# stremio-sensitivity-addon
A Stremio addon that surfaces **crowdsourced content-sensitivity flags** (animal death, sexual violence, suicide, flashing lights for photosensitive epilepsy, ~200 more) from **DoesTheDogDie.com (DTDD)**, shown *before* the user presses play. The user pins the triggers that matter to them once, and every title they browse through is pre-flagged.
